# Security policy

GoldenEye Metal is an early research project and does not currently publish supported releases.

Please report a suspected vulnerability privately through the repository's security-advisory
feature. Include the affected commit, impact, reproduction steps, and any proposed mitigation. Do
not open a public issue for an unpatched vulnerability or include proprietary game data in a
report.

Security fixes are handled on a best-effort basis. A report is not eligible for a bounty unless a
separate written program explicitly says otherwise.
